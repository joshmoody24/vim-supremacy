# Vim Supremacy

Vi mode everywhere

## Setup Instructions

### Neovim settings

Requires vim-plug.

Put `coc-settings.json` and `init.vim` in the Neovim config location, probably `~/.config/nvim/`

### Bash settings

Get vi mode by doing `git apply patch.bashrc`

### Tmux settings

Put `.tmux.conf` in `~`
